# Agent Loop

This folder contains the multi-agent finding queue.

Do not manually edit generated queue files.

Use `agentq`:

```bash
agentq append ...
agentq claim ...
agentq fix ...
agentq verify ...
agentq render
agentq doctor
```

Status flow:

```text
open -> claimed -> fixed_pending_verification -> verified
```

Auditors create findings.
Fixers fix one finding at a time.
Verifiers close findings.

## Source Of Truth

- Findings: `docs/agent-loop/findings/*.md`
- Event trails: `docs/agent-loop/events/*.jsonl`
- Generated queue views: `docs/agent-loop/queues/*.md`

Queue Markdown is regenerated by `agentq render`; do not edit it by hand.

## Local Tool

During development, use either wrapper:

```bash
./scripts/agentq.sh doctor
```

```powershell
./scripts/agentq.ps1 doctor
```

Or run the project directly:

```bash
dotnet run --project tools/AgentQueue -- doctor
```
